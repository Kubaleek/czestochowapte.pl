## Polskie Towarzystwo Ekonomiczne - Oddział w Częstochowie

<p>
czestochowapte.pl to szkolny projekt - aplikacja webowa oparta głównie na wyglądzie strony zarządu krajowego pte.pl. Została stworzona przy użyciu Next.js, TypeScript, Tailwind CSS, GraphQL oraz ShadCN. Aplikacja jest w pełni zintegrowana z CMS WordPress, co umożliwia dynamiczne zarządzanie treściami, w tym nawigacją, stopką oraz stronami.
</p>
